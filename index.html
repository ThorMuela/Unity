🟩 Unity Loader (Mod para Minecraft)

Unity es un loader para Minecraft que permite crear menús personalizados, cargar mods de forma modular y controlar el flujo del juego.

📥 Descarga

  • Descargar Unity Loader (.zip)

📋 Requisitos

  • Minecraft 1.20.1 o superior
  • Compatible con: NeoForge / Forge / Fabric
  • Java 17 o superior

🛠️ Características

  • Menús interactivos personalizados
  • Carga modular de mods
  • Sistema preparado para compatibilidad cruzada
